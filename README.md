# ZORAGO 🚀

**Global 18+ Female-Focused Live Streaming Platform**

Built with ❤️ from Addis Ababa, Ethiopia

---

## ✨ Features

- 🔐 Secure Authentication (18+ verification)
- 👤 User Profiles & Following
- 🎥 Live Streaming with Agora SDK
- 💬 Real-time Chat
- 💰 Tipping & Monetization
- 🔍 Discover & Search

---

## 🛠️ Tech Stack

| Component | Technology |
|-----------|-----------|
| **Frontend** | Next.js + React + TailwindCSS |
| **Backend** | Node.js + Express |
| **Database** | Firebase Realtime DB |
| **Video** | Agora SDK |
| **Payments** | Stripe |
| **Hosting** | Vercel + Railway |
